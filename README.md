# Data Analytics Practice

This repository contains SQL exercises and small data analysis problems used to practice core data analytics skills.

## Skills Practiced

* SQL Joins
* Subqueries
* Aggregations
* Data filtering
* Analytical queries

## Dataset

The exercises use a **library management database** with the following tables:

* employees
* books
* categories
* authors
* loans
* fines

## Example Problems

1. Find the employee(s) with the earliest hiring year.
2. Find the most recently published book in each category.
3. Find books with more copies than the average of their category.
4. Find the book that generated the highest total amount of fines.
